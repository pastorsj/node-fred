# Example

## Using Promises

```javascript
import Fred from 'node-fred';

const fred = new Fred(API_KEY);

function getCategory(categoryID) {
    fred.categories
        .getCategory(125)
        .then((res) => {
            console.log('Category', res);
        })
        .catch((err) => {
            console.error('Error', err);
        });
}
getCategory(125);
```
